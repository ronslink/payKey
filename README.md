# PayKey - Domestic Worker Management Platform

## Overview

PayKey is a cross-platform mobile application designed specifically for expats in Kenya to efficiently manage domestic workers. The app operates on a subscription-based SaaS model, prioritizing ease of use and effectiveness as its core value proposition.

## Key Features

- **Subscription Management**: Multi-tier pricing with free trials
- **Payment Processing**: M-Pesa integration with bulk payments
- **HR Management**: Employee profiles, leave tracking, performance management
- **Tax Compliance**: PAYE, NSSF, NHIF calculations and reporting
- **Communication**: In-app messaging with SMS fallback
- **Multi-language Support**: English, Swahili, French

## Target Market

- Expats living in Kenya
- Small to medium businesses with domestic staff
- Property management companies
- NGOs and international organizations

## Business Model

- **Free Tier**: 1 worker, basic features
- **Basic Tier**: Up to 3 workers ($15/month)
- **Professional Tier**: Up to 10 workers, advanced features ($45/month)
- **Enterprise Tier**: Unlimited workers, white-label options ($120/month)

## Technology Stack

- **Frontend**: React Native
- **Backend**: Node.js with microservices
- **Database**: PostgreSQL with Redis caching
- **Payments**: Stripe + M-Pesa Daraja API
- **Infrastructure**: AWS with Kenya region

## Development Phases

1. **Phase 1 - MVP** (3 months): Core subscription, basic payments, employee management
2. **Phase 2 - Enhancement** (2 months): Advanced HR, compliance, messaging
3. **Phase 3 - Scale** (2 months): Enterprise features, integrations, analytics
